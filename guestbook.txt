Создание гостевой книги на PHP 


1. сделать файл формы которыя состоит из:
    a: Текстового поля (отзыв)
    b: Текстового поля (Имя и Фамилия)
    c: Кнопки отправки формы
2. написать файл скрипта, который обрабатывает форму (POST) и добавляет данные в базу
3. добавить таблицу в базе данных (gestbook)
4. Сделать вывод отзывов на страницу (можно на существующую где форма)
5. сделать базовое оформление CSS    
6. Захостить на сервер и прислать ссылку
